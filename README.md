# Desarollo de Software/Configuracion del Entorno
Este repositorio contiene el avance del proyecto correspondiente a la asignatura Lógica de Programación II.
El software desarrollado es el juego Piedra, Papel o Tijera, el cual permite aplicar los conceptos de algoritmos, diagramas de flujo y lógica de programación.
